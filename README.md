# Studymate-v2

Studymate v2 is a full-stack application designed to help students generate notes from uploaded documents using NLP processing. The app features a React frontend and a Flask backend with document parsing and text analysis capabilities.

Features

- Upload documents for automatic note generation
- NLP-based text processing for extracting key points
- Clean and modern React user interface
- Flask backend with secure file handling and API endpoints
- Modular backend structure with expandable processors

Tech Stack

| Area | Technology |
|------|------------|
| Frontend | React, JavaScript |
| Backend | Python, Flask |
| NLP | Custom text parsing + NLP processing |
